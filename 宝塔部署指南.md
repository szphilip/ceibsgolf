# 🎯 宝塔面板 + GitHub 部署完整指南

## 📋 部署流程概览

```
本地修改 → Git推送 → GitHub仓库 → 宝塔拉取 → 服务器更新
  ↓          ↓           ↓            ↓           ↓
编辑代码   git push   代码托管    点击按钮    自动部署
```

---

## 第一步：准备GitHub仓库

### 1. 创建GitHub账号

如果还没有：访问 https://github.com 注册

### 2. 创建新仓库

1. **登录GitHub**
2. **点击右上角 "+" → New repository**
3. **填写信息**：
   - Repository name: `ceibsgolf`
   - Description: `CEIBS Suzhou Golf Team Website`
   - Public（公开）或 Private（私有）
   - ✅ 勾选 "Add a README file"
4. **点击 Create repository**

### 3. 获取仓库地址

创建后会看到仓库地址：
```
https://github.com/yourusername/ceibsgolf.git
```

复制这个地址，稍后会用到。

---

## 第二步：上传代码到GitHub

### 方法A：使用GitHub网页上传（最简单）

1. **打开GitHub仓库页面**
2. **点击 "Add file" → "Upload files"**
3. **拖拽以下文件**：
   - `index.html`（重命名为 `index_with_server_storage.html`）
   - `api.php`
   - `.gitignore`
   - `README.md`
4. **填写提交信息**：`Initial commit`
5. **点击 "Commit changes"**

### 方法B：使用Git命令行（推荐）

**Windows用户先安装Git**：
- 下载：https://git-scm.com/download/win
- 安装后右键 → Git Bash Here

**Mac用户**：
终端已自带Git

**操作步骤**：

```bash
# 1. 进入文件所在目录
cd /path/to/your/files

# 2. 初始化Git
git init

# 3. 添加所有文件
git add .

# 4. 提交
git commit -m "Initial commit"

# 5. 关联远程仓库
git remote add origin https://github.com/yourusername/ceibsgolf.git

# 6. 推送到GitHub
git branch -M main
git push -u origin main
```

**输入GitHub用户名和密码**（如果需要）

**✅ 完成！刷新GitHub页面应该能看到文件**

## 第三步：宝塔面板配置

### 前提条件

- ✅ 已安装宝塔面板
- ✅ 域名已解析到服务器
- ✅ 宝塔版本 ≥ 7.0

### 1. 创建网站

#### A. 添加站点

1. **宝塔面板 → 网站 → 添加站点**

2. **填写信息**：
   ```
   域名：www.ceibsgolf.cn
   根目录：/www/wwwroot/ceibsgolf
   FTP：不创建
   数据库：不创建
   PHP版本：7.4 或 8.0
   ```

3. **点击"提交"**

#### B. 配置目录

如果目录已存在内容：

```bash
# SSH登录或使用宝塔终端
cd /www/wwwroot
rm -rf ceibsgolf  # 删除旧目录
mkdir ceibsgolf
```

### 2. 配置Git部署

#### A. 进入Git设置

1. **网站列表 → 找到 www.ceibsgolf.cn**
2. **点击"设置"**
3. **左侧菜单 → Git**

#### B. 添加Git仓库

**仓库地址**：
```
https://github.com/yourusername/ceibsgolf.git
```

**分支名称**：
```
main
```

**目标目录**：
```
/www/wwwroot/ceibsgolf
```

**注意**：
- ✅ 使用HTTPS地址（不是SSH）
- ✅ 分支名确认是 `main` 不是 `master`

#### C. 首次拉取

点击 **"拉取"** 按钮

等待几秒，应该显示：
```
✅ 拉取成功
```

查看文件：
```bash
ls -la /www/wwwroot/ceibsgolf
```

应该看到：
```
index.html
api.php
.gitignore
README.md
```

### 3. 配置SSL证书

#### A. 申请Let's Encrypt免费证书

1. **网站设置 → SSL**
2. **Let's Encrypt**
3. **填写域名**：
   ```
   www.ceibsgolf.cn
   ceibsgolf.cn
   ```
4. **填写邮箱**（用于证书过期提醒）
5. **点击"申请"**

等待1-2分钟，显示 **"证书申请成功"**

#### B. 启用HTTPS

1. **部署证书**（自动）
2. **强制HTTPS**：开启
3. **HSTS**：开启（可选）

#### C. 测试

访问：`http://www.ceibsgolf.cn`

应该自动跳转到：`https://www.ceibsgolf.cn`

地址栏显示🔒图标

### 4. 修改API密码

**非常重要！必须修改密码！**

#### A. 编辑api.php

方法1：宝塔文件管理器
```
文件 → /www/wwwroot/ceibsgolf/api.php → 编辑
```

方法2：SSH命令行
```bash
cd /www/wwwroot/ceibsgolf
nano api.php
```

**找到第16行**：
```php
define('API_PASSWORD', 'CHANGE_ME_IN_PRODUCTION');
```

**改为强密码**：
```php

```

保存（Ctrl+O，然后Ctrl+X）

#### B. 编辑index.html

**找到约1583行**：
```javascript
const SERVER_CONFIG = {
  apiUrl: '/api.php',
  password: 'CHANGE_ME_IN_PRODUCTION',
```

**改为相同密码**：
```javascript
  password: 'CeibsGolf@2024!Secure123',
```

**⚠️ 两个文件的密码必须一致！**

### 5. 创建数据目录

#### 方法1：宝塔文件管理器

1. **文件 → /www/wwwroot/ceibsgolf**
2. **新建目录 → `data`**
3. **进入data → 新建目录 → `backups`**

5. **右键 data → 所有者 → www**

```bash

mkdir -p data/backups
chmod 755 data
chmod 755 data/backups
chown -R www:www data
```

## 第四步：日常更新流程

### 完整流程

```bash
# 1. 本地修改文件
# 用任何编辑器修改 index.html 或其他文件

# 2. 提交到Git
git add .
git commit -m "更新：添加新功能"

# 3. 推送到GitHub
git push origin main

# 4. 宝塔部署
# 打开宝塔面板 → 网站 → 设置 → Git → 点击"拉取"
# 等待几秒 → 显示"拉取成功"

# 5. 测试
# 访问网站确认更新生效
```

### 一键更新脚本（可选）

创建 `update.sh`：

```bash
#!/bin/bash
echo "📝 添加更改..."
git add .

echo "💬 输入提交信息："
read message
git commit -m "$message"

echo "🚀 推送到GitHub..."
git push origin main

echo "✅ 完成！请在宝塔面板点击'拉取'按钮"
```

使用：
```bash
chmod +x update.sh
./update.sh
```

## 第五步：高级配置（可选）

### 1. 自动部署Webhook

#### GitHub配置

1. **仓库 → Settings → Webhooks → Add webhook**
2. **Payload URL**：
   ```
   https://www.ceibsgolf.cn/webhook.php
   ```
3. **Content type**：`application/json`
4. **Secret**：设置一个密钥（例如：`ceibsgolf_webhook_2024`）
5. **事件**：选择 `Just the push event`
6. **Active**：✅
7. **Add webhook**

#### 创建webhook.php

在服务器创建 `/www/wwwroot/ceibsgolf/webhook.php`：

```php
<?php
// Webhook自动部署脚本
$secret = 'ceibsgolf_webhook_2024'; // 与GitHub设置相同

// 验证签名
$signature = $_SERVER['HTTP_X_HUB_SIGNATURE_256'] ?? '';
$payload = file_get_contents('php://input');
$hash = 'sha256=' . hash_hmac('sha256', $payload, $secret);

if (!hash_equals($signature, $hash)) {
    http_response_code(403);
    exit('Invalid signature');
}

// 执行Git拉取
$output = shell_exec('cd /www/wwwroot/ceibsgolf && git pull origin main 2>&1');

// 记录日志
file_put_contents(
    'webhook.log',
    date('Y-m-d H:i:s') . " - Deploy triggered\n" . $output . "\n\n",
    FILE_APPEND
);

echo json_encode(['status' => 'success', 'message' => 'Deployed']);
?>
```

**现在每次Git推送，服务器会自动更新！**

### 2. 定期备份到云盘

使用宝塔计划任务：

```bash
# 每天凌晨2点备份

```

## 🆘 故障排查

### 问题1：宝塔Git拉取失败

**错误提示**：`fatal: not a git repository`

**解决**：
```bash
cd /www/wwwroot/ceibsgolf
git init
git remote add origin https://github.com/yourusername/ceibsgolf.git
git pull origin main
```

### 问题2：权限被拒绝

**错误提示**：`Permission denied`

**解决**：
```bash
chown -R www:www /www/wwwroot/ceibsgolf
chmod -R 755 /www/wwwroot/ceibsgolf
```

### 问题3：API返回404

**原因**：PHP未正确配置

**解决**：
1. 宝塔 → 网站 → 设置 → PHP版本
2. 确认选择了7.4或8.0
3. 重启PHP

### 问题4：data目录无法写入

**解决**：
```bash
cd /www/wwwroot/ceibsgolf
chmod 755 data
chown www:www data
```

### 问题5：Webhook不工作

**检查**：
1. 访问 `https://www.ceibsgolf.cn/webhook.php`（应该返回错误，但不是404）
2. 检查 GitHub Webhooks → Recent Deliveries
3. 查看响应状态码
